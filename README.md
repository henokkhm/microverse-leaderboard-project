# Leaderboard <a name="about-project"></a>

This project is the Leaderboard project for Module 2 - Week 4 of the Full-Stack Developer program offered by Microverse.It uses the basic Webpack template, which we use as a starting point for some Microverse projects. It also sets up linters for HTML, CSS and JavaScript.

### Key Features <a name="key-features"></a>

- **Webpack:** is used to bundle JavaScript files.
- **HtmlWebpackPlugin:** is used to automatically generate the index.html file in the /dist directory.
- **webpack-dev-server :** improves developer experience by watching your source files, automatically compiling distribution files and refreshing the browser every time you *save* changes in the source code.

## 💻 Getting Started <a name="getting-started"></a>

To run the project, you should first clone this repository. 

### Prerequisites

In order to run this app you need:

- <a href="https://nodejs.org/en/download">Node.js</a>
- <a href="https://docs.npmjs.com/downloading-and-installing-node-js-and-npm">npm</a>

### Setup

Clone this repository to your desired folder:

```sh
  cd desired-folder
  git clone https://github.com/henokkhm/microverse-leaderboard-project.git
```

### Install

Install the dependencies with:

```sh
  cd microverse-leaderboard-project
  npm install
```

### Usage

To run the project, run `npm run dev`

To run the linters, run `npm run lint`

To generate static assets using Webpack, run `npm run build`


### Run tests

There are currently no tests in this project.

### Deployment

You can deploy the static assets generated by Webpack using any web-hosting service you choose.
